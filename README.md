# Commodities Management System (Frontend UI)

A frontend prototype implementing role-based access for a commodities management system.

## Features
- Login with role selection (Manager / Store Keeper)
- Manager-only Dashboard
- Products page for both roles
- Role-based menu restriction
- Light UI with modern glassmorphism design

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone repository
2. Open `index.html` in any browser
3. Login and explore role-based UI

## Sample Roles
Manager → Access Dashboard + Products  
Store Keeper → Access Products only
